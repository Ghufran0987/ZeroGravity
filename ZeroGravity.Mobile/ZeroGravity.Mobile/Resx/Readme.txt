
Hinweise:
- AppResources.resx ist die Default Ressourcendatei, aus der VisualStudio dann die Klasse AppResources automatisch generiert.
  Sie enthält die selben Einträge wie AppResources-en-US.resx (bzw. sollte enthalten), weil in AssemblyInfo.cs das 
  NeutralResourcesLanguage Attribut auf en-US definiert wird.

- AppResources.de-DE.resx funktioniert nicht
=> deshalb AppResources.de.resx
